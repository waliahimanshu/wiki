## Code Solution:

```js
function booWho(bool) {
  return typeof bool === 'boolean';
}
```

# Code Explanation:

Uses the operator `typeof` to check if the variable is a boolean.
If it is, it will return `true`. Otherwise, if it is any other type it will return `false`.
